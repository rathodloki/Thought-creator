# ✨ Thought Creator ✨

A fun web app to write, style, and download your thoughts as images.

## 🚀 Features

- 📝 Write your thoughts
- 🎨 Style your text
- 📸 Download as an image
- 🔄 Refresh random background
- 🎵 Listen to music while writing
